# Interactive 3D Text Animation

![Three.js](https://img.shields.io/badge/Three.js-black?style=for-the-badge&logo=three.js&logoColor=white)
![GSAP](https://img.shields.io/badge/GSAP-88CE02?style=for-the-badge&logo=greensock&logoColor=black)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

An interactive 3D particle visualization built with Three.js and GSAP. This project features a dynamic particle system that can smoothly morph between a spherical distribution and custom text input, complete with interactive controls and fluid animations.

## âœ¨ Features

- **Interactive 3D Sphere**: A mesmerizing particle sphere that responds to mouse interactions.
- **Text Morphing**: Type any text and watch the particles fluidly rearrange themselves to form the letters.
- **Dynamic Color System**: 
  - Real-time color customization via a sleek color picker.
  - Adaptive HSL-based color themes that maintain visual depth.
- **Mouse Interaction**:
  - Rotate the sphere by dragging.
  - Text tracks mouse movement for a 3D parallax effect.
- **Responsive Design**: Adapts to different screen sizes and devices.

## ğŸš€ Getting Started

### Prerequisites

You don't need any complex build tools or package managers to run this project. A modern web browser is all that's required.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ayush-ranjan-cmaug25/Interactive-3D-Text-Animation.git
    ```
2.  **Open the project**:
    Simply open the `index.html` file in your preferred web browser.

### Usage

1.  **Explore**: Move your mouse to interact with the particle sphere. click and drag to rotate it.
2.  **Type**: Enter text into the input field at the bottom of the screen and press "Enter" or click the "Create" button.
3.  **Colorize**: Click the paint palette icon to choose a new base color for the particles.
4.  **Reset**: Click the "Reset" button to return the particles to their spherical form.
